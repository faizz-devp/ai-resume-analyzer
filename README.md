# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using **Python**, **Streamlit**, and **Google Gemini AI**.

## 🚀 Live Demo

🔗 https://ai-resume-analyzer-wdswug5dejqtbkvtw67sb5.streamlit.app

## 📷 Features

- 📄 Upload PDF resumes
- 📝 Extract resume text
- 🎯 Compare resumes with job descriptions
- 📊 Calculate ATS match score
- ❌ Detect missing skills
- 🤖 Generate AI-powered resume feedback using Google Gemini
- 👀 Preview extracted resume text

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API
- PyMuPDF (fitz)
- python-dotenv
- Git
- GitHub

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/faizz-devp/ai-resume-analyzer.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
